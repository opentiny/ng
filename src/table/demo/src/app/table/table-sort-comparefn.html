描述

可使用tiHeadSort组件的compareFn接口来自定义所在列的排序规则。

注意：使用 tiOverflow 组件，请导入 TiOverflowModule。

示例

{{columns[0].title}} （此列用了自定义排序规则） {{columns[1].title}} {{columns[2].title}} （此列用了自定义排序规则） {{columns[3].title}} {{columns[4].title}}
