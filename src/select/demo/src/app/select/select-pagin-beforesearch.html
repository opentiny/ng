描述

1.Select的下拉选项如果需要分页显示，需要通过footer将分页添加到下拉面板，如果footer功能异常，参考“模板”示例。

2.分页结合后台搜索的功能，需要开发者控制分页的 currentPage ，并且将对应页码的options数据传入组件。

导入

import {{ '{' }} TiSelectModule, TiPaginationModule {{ '}' }} from '@opentiny/ng';
